Rules Entries must have an image as proof with the following details visible: - Route on map (except for swimming) - Distance travelled - Duration The sports can be performed either outdoors or indoors, but you have to be moving. Thus, no treadmills or spinning etc. The coefficients for different sports are as follows: - Swimming: 5 - Running or walking: 1.75 - Cross-country skiing: 1 - Ice skating: 0.5 - Cycling: 0.5 Minimum distance: 0.5km (except for swimming)
