Rules Entries must have an image as proof with the following details visible: - Route on map (except for swimming/rowing) - Distance travelled - Duration - Date of the activity The sports can be performed either outdoors or indoors, but you have to be moving. Thus, no treadmills or spinning etc. (This does not apply to rowing, which can be done on a rowing machine.) The coefficients for different sports are as follows: - Swimming: 4 - Running or walking: 1.5 - Cross-country skiing: 1 - Ice skating: 1 - Cycling: 0.75 - Rowing: 1 Minimum distance: 0.5km (except for swimming)
