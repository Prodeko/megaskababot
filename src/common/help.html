Welcome to Megaskababot! Megaskababot helps you record and keep track of your entries in the competition between Athene, IK, TiK, MK, DG, PT, Inkubio, FK, AK, AS and Prodeko. If you have any problems with the bot, please contact @heinikoivu on Telegram. Instructions The bot has the following commands: - /start - Start the bot - /entry - Add a new entry - /entries - Get a list of your own entries - /removelatest - Remove your latest entry - /help - Show this message - /rules - Show the rules Adding a new entry You can add a new entry by typing /entry. It will ask you the distance, sport and a photo as a proof. The following sports are accepted: cross-country skiing, walking, running, cycling, ice skating, rowing, and swimming. The photo must include the date and time, the route and the distance traveled from a sports tracking application. Transparency Megaskababot's source code can be viewed here.
