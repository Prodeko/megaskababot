Welcome to MEGASCIBABOT! MEGASCIBABOT helps you record and keep track of your entries in the competition between Prodeko&Athene, Tietokilta&Data Guild and Fyysikkokilta. If you have any problems with the bot, pelase contact @mediakeisari on tg. Instructions The bot has the following commands: /start - Start the bot /entry - Add a new entry /entries - Get a list of your own entries /removelatest - Remove your latest entry /help - Get help Adding a new entry You can add a new entry by typing /entry. It will ask you the distance, sport and a photo as a proof. The sport must be either run/walk, ski or skate. The photo must include the route and the distance traveled from a sports tracking application. Transparency MEGASCIBABOT's source code can be viewed here
