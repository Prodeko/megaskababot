Welcome to Megaskababot! Megaskababot helps you record and keep track of your entries in the competition between Athene, Indecs, Man@ger and Prodeko. If you have any problems with the bot, please contact @mUusitalo on Telegram. Instructions The bot has the following commands: - /start - Start the bot - /entry - Add a new entry - /entries - Get a list of your own entries - /removelatest - Remove your latest entry - /help - Show this message - /rules - Show the rules Adding a new entry You can add a new entry by typing /entry. It will ask you the distance, sport and a photo as a proof. The following sports are accepted: skiing, walking, running, cycling, ice skating, and swimming. The photo must include the route and the distance traveled from a sports tracking application. Transparency Megaskababot's source code can be viewed here.
